Plugin adds custom logo widget - images with links.

== Installation ==

1. Unpack and upload 'logo-widget' directory to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Theme' -> 'Widgets' and use 'Logo widget' as you want!
